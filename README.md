# eslint-plugin-github

* [Package Requirements](/docs/package-requirements.md)

## Installation

``` sh
$ npm install --save-dev eslint
$ npm install --save-dev eslint-plugin-github
```

``` js
// .eslintrc.json
{
  "extends": [
    "plugin:github/recommended",
    "plugin:github/browser",
    "plugin:github/es6"
  ]
}
```
